#### 2. 运动员管理规则

##### 2.1 注册要求

- **注册号管理**:
  - 所有注册运动员将获得注册号
  - 仅有注册号的运动员允许参赛

- **夏季赛季**: 每年6月1日至10月15日（室内雪场赛事）
- **积分有效期**: 24个月滚动计算（v2.0新增）

### 🧮 核心积分计算规则 ⭐ v2.0重大升级

#### 4.1 基础比赛积分计算公式
```
P = (F × Tx/To) - F

参数说明:
P = 基础比赛积分
To = 本场比赛冠军用时（秒）
Tx = 该运动员用时（秒）
F = 小项系数
```

**项目系数(F值)**:
- 滑降(DH): 1250
- 回转(SL): 730
- 大回转(GS): 1010
- 超级大回转(SG): 1190
- 全能(AC): 1360

#### 4.2 综合积分计算公式 ⭐ v2.0官方标准
```
总积分 = 基础积分 × 赛事系数 × 质量系数 × 人数系数 + 附加分

赛事系数（官方v2.0标准）：
- A级赛事：1.0（全国锦标赛、冬运会等）
- B级赛事：0.8（青年冠军赛、巡回赛分站等）
- C级赛事：0.6（省级锦标赛、区域邀请赛）
- D级赛事：0.4（市级比赛、俱乐部赛事）

质量系数：质量系数 = 1 + (前8名选手平均中国积分/500)

人数系数（v2.0精确分档）：
- 8-15人：×0.8
- 16-31人：×1.0
- 32-63人：×1.1
- 64-127人：×1.2
- 128-255人：×1.3
- 256人以上：×1.4

附加分体系：
- 连续完赛加分：单日双项+3分，连续两站全勤+5分，赛季全勤+10分
- 战胜高手加分：(被击败选手积分-自己积分)/100×赛事级别系数，最高10分
- 突破加分：刷新个人最佳+5分，项目首次完赛+3分，年度进步奖+8分
```

#### 4.3 计算示例详解 ⭐ v2.0新示例

**基础积分计算示例：**
```
示例: 运动员在B级大回转比赛中的积分计算
- 冠军用时(To) = 65.43秒
- 运动员用时(Tx) = 67.89秒
- 项目系数(F) = 1010

基础积分计算:
P = (1010 × 67.89/65.43) - 1010 = 38.00分
```

**综合积分计算示例（v2.0官方标准）：**
```
实际比赛场景：B级大回转比赛，65人参赛
- 基础积分: 38.00分
- 赛事系数: 0.8 (B级赛事标准)
- 质量系数: 1 + (前8名平均积分180分/500) = 1.36
- 人数系数: 1.2 (65人，适用64-127人档)
- 附加分: 3分 (项目首次完赛) + 2分 (战胜高手) = 5分

总积分 = 38.00 × 0.8 × 1.36 × 1.2 + 5 = 54.58分

青少年运动员额外计算（U15组别）：
U15最终积分 = 54.58 × 0.85 (年龄系数) + 2.0 (进步奖励) = 48.39分
```

#### 4.2 基础积分列表(BL)计算

##### 4.2.1 标准BL计算
- 根据上赛季各小项最好成绩计算
- **两个成绩**: BL = (最好成绩1 + 最好成绩2) ÷ 2
- **单次成绩**: BL = 成绩 × 1.15 (中国优化: 比国际1.20降低)
- **无成绩**: BL = 上年BL × 1.40 (中国优化: 比国际1.50温和)
- **新运动员**: BL = 项目国内平均水平 × 1.20

#### 4.3 常规积分列表(NL)规则
- 体现当前积分周期中运动员各小项最好的两次成绩平均值
- 只有优于BL时才更新
- **更新周期**: 每21天一次 (中国优化: 比国际14天适配赛事频率)
- **积分精度**: 保留到小数点后2位

**最大积分保护值(中国优化)**:
- 滑降: 300分 (国际330分)
- 超级大回转: 250分 (国际270分)
- 大回转: 200分 (国际220分)
- 回转: 150分 (国际165分)
- 全能: 250分 (国际270分)

#### 4.4 判罚分(Penalty)计算 ⭐ 中国核心优化
```
Penalty = (Sum A + Sum B - Sum C) ÷ 12

参数说明:
Sum A = 前10名中最好5名的中国积分总和
Sum B = 全部参赛者中最好5名的中国积分总和
Sum C = 最好5名的比赛积分总和

中国优化: 分母调整为12 (国际为10)，适配中国参赛规模
特殊处理: Penalty上限不超过50分
动态调整: 参赛人数不足10人时采用动态调整算法
```

#### 4.5 伤病保护机制

##### 4.5.1 基础保护规则
- 须提交三甲医院诊断证明
- 伤病期超过60天可申请
- **保护调整**: BL积分上调8% (中国优化: 比国际10%更稳健)
- **保护期限**: 最长10个月
- **审核时间**: 5个工作日内完成

##### 4.5.2 创新保护模式 ⭐ 中国特色
- **分阶段保护申请**: 根据恢复进度动态调整
  - 第一阶段: 初期伤病，BL积分上调8%
  - 第二阶段: 康复期，BL积分上调5%
  - 第三阶段: 恢复期，BL积分上调3%
- **积分冻结选项**: 重大伤病期间积分完全不变
  - 适用条件: 住院治疗超过30天的重大伤病
  - 冻结期限: 最长6个月
  - 申请材料: 三甲医院住院证明+手术记录
- **灵活申请机制**: 考虑中国运动员训练周期特点
  - 支持赛季中途申请
  - 支持多次申请(每年最多2次)
  - 简化审核流程(5个工作日内完成)
